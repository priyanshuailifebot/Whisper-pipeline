## Compute Infrastructure & Capacity

### National IndiaAI Compute Capacity Mission

#### Overview
10,000+ GPU ecosystem through public-private partnerships, offering AI services and resources.

**Pillars:**
1. **Infrastructure Development:** Scalable AI ecosystem creation
2. **Service Provision:** AI computing access and tools
3. **Resource Optimization:** Efficient resource utilization
4. **Innovation Enablement:** Research and development support

### Gujarat Technical Infrastructure

#### Data Centers
- **Tier Requirements:** Tier III or above certification
- **Area Specifications:** Minimum 4,000 sq. ft. built-up area
- **Capacity Metrics:** 150+ racks OR 900 KW power consumption OR 2,00,000+ processor cores
- **Incentives:** CAPEX/OPEX support as per policy guidelines

#### GPU Infrastructure
- **Standalone GPUs:** Separately installed from CPU (embedded GPUs not eligible)
- **Deep Tech Applications:** Essential for AI, ML, Generative AI processing
- **Extended Depreciation:** 5-year depreciation period for deep tech hardware

#### Cloud Infrastructure
- **Bandwidth & Cloud Rental Caps:** 35% of eligible OPEX or specified ceilings
- **Shared Infrastructure Support:** 35% subsidy for bandwidth and cloud costs
- **Private Cloud Integration:** Support for cloud platform procurement
